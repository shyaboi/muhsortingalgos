This is a sorting algorithm API.


User Story

You want to sort a list of numbers. You want to see which sorting algorithm will get your information sorted in the least amount of iterations.


Usage

This API in its current version, can be used via form data in a POST request


Post Man


To use this API with Post man, use the 'form data' in the body type.


The fist key of the form will be labeled 'arrToSort'. For the value, you can currently enter any amount of numbers seperated by spaces.

The second key of the form is for a random number of indices. This field is required, but only read if the first form is blank. The key in the form should be entered as 'randomIndices', and the value can be any integer.

Example POST

Result

A succsessful post to /bubbleSort will result in an array of arrays with every itteration it took to sort the given numbers or indices
